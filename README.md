Special key-bind instuction below:\n

  -'mouse whee up'  jump\n
  -'mouse3'         switch to flash\n
  -'mouse4'         speak\n
  -'x'              jump throw\n
  -'c'              switch to smoke\n
  -'F4'             crosshair 1\n
  -'F5'             crosshair 2\n
  -'F6'             crosshair 3\n
  -'F8'             re-execuate joe.cfg with quick 'q'\n
  -'F9'             unbind quick 'q'\n

note: quick 'q' means press 'q' for twice by command in consolo
